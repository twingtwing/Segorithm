## Bipartite Graph
그래프의 정점의 집합을 둘로 분할하여,
각 집합에 속한 정점끼리는 서로 인접하지 않도록 분할할 수 있을 때,
그러한 그래프를 특별히 이분 그래프 (Bipartite Graph)

> 이분 그래프를 판별하는 알고리즘 중 하나인 DFS (깊이 우선 탐색)를 활용한 색칠 (Coloring) 알고리즘
